🔌 ปลั๊กอินบอท Telegram / AI LOCAL

🔑 READ token (เช็คราคา)
🔐 ADMIN token (แก้ไข/สั่งงาน)
แยกโทเคน 2 ชุด: READ เห็นเฉพาะราคาที่มีผลแล้ว (ไม่เห็นทุน/รหัสลับ) · ADMIN แก้ราคา-เผยแพร่ได้เหมือนนั่งหน้าจอ · ไฟล์เซิร์ฟเวอร์ตัวอย่าง: telegram-bridge-server.example.js (รันด้วย Node.js บนเครื่องคุณ)
คำสั่ง API (JSON ผ่าน WebSocket)
price.check {q}🔑 เช็คราคา (สูงสุด 10 รายการ)
price.list {q}🔑 รายการราคา (สูงสุด 60)
admin.price.check {q}🔐 เช็คราคา + ทุน/Margin/ร่างล่าสุด
admin.price.set {q, field, value}🔐 แก้ราคา (field: retail/B/A/S/cost)
admin.cell.set {ref, value}🔐 แก้ช่องใดก็ได้ เช่น H9 = 1850 หรือสูตร
admin.update.publish {effectiveAt?}🔐 เผยแพร่ราคาเข้า DB (ตั้งเวลาได้)
admin.changes🔐 แถวที่มีการปรับรอบนี้
